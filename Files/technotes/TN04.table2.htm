User Interface Color Table
NSBasic Tech Note 12
# Object Description
0 UIObjectFrame Color for the border of user interface objects (such as command buttons, push buttons, selector triggers, menus, arrows checkboxes, and other controls).
1 UIObjectFill The background color for a solid or "filled" user interface object.
Note that UI objects in tables use the UIField... colors instead of the UIObject... colors.
2 UIObjectForeground The color for foreground items (such as labels or graphics) in a user interface object.
3 UIObjectSelectedFill The background color of the currently selected user interface object, whether that object is solid or not.
4 UIObjectSelectedForeground The color of foreground items in a selected user interface object.
5 UIMenuFrame The color of the border around the menu.
6 UIMenuFill The background color of a menu item.
7 UIMenuForeground The color of the menu's text.
8 UIMenuSelectedFill The background color of a selected menu item.
9 UIMenuSelectedForeground The color of the text of a selected menu item.
10 UIFieldBackground The background color of an editable text field.
11 UIFieldText The color of the text in the editable field.
12 UIFieldTextLines The color of underlines in an editable field.
13 UIFieldCaret The color of the cursor in an editable text field.
14 UIFieldTextHighlightBackground The background color for selected text in an editable text field.
15 UIFieldTextHighlightForeground The color of the selected text in an editable text field.
16 UIFieldFepRawText Color used for unconverted text in the inline conversion area when a FEP is used as a text input method (for example, on Japanese devices).
If the FEP colors are identical to field colors, unconverted text has a solid underline.
17 UIFieldFepRawBackground The background color for unconverted text in the inline conversion area when a FEP is used as a text input method.
If the FEP colors are identical to field colors, unconverted text has a solid underline.
18 UIFieldFepConvertedText Color used for converted text in the inline conversion area when a FEP is used as a text input method (for example, on Japanese devices).
If the FEP colors are identical to field colors, converted text has a double-thick underline.
19 UIFieldFepConvertedBackground The background color used for converted text in the inline conversion area.
If the FEP colors are identical to field colors, converted text has a double-thick underline.
20 UIFieldFepUnderline The color used for underlines in the inline conversion area.
21 UIFormFrame The color of the border and titlebar on a form.
22 UIFormFill The background color of a form. White is recommended for this value.
23 UIDialogFrame The color of a border and titlebar on a modal form.
24 UIDialogFill The background color of a modal form.
25 UIAlertFrame The color of the border and titlebar on an alert panel.
26 UIAlertFill The background color of an alert panel.
27 UIOK The color for an informational icon.
28 UICaution The color for a caution icon.
29 UIWarning The color for a warning icon.
Palm OSÆ does not currently use the UIOK, UICaution, and UIWarning constants.
30 UILastColorTableEntry Placeholder to indicate end of enum.

Definitions:

FEP
The current most widely-used methodology for entering Japanese data is via built-in or third-party software which allows text to be entered by typing the word phonetically at the keyboard (usually using English phonetics or "Romaji" rather than Kana). The user types the phonetic equivalent of the relevant Kanji and is provided with a list of several options having the same pronunciation (rather like searching the selections offered by a conventional spell checker). Either the default choice, or one of the options on the list, is selected by the user.

This input software, called a front-end processor, or FEP (pronounced "fep"), must be able to scan through thousands of characters, usually in memory. While this sounds like an impossibly slow process, Japanese users become very adept and the software is very fast.
