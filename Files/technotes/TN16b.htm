1. Opening an IR port

To open an IR port for serial communications, 

   res=serialopen(32769,9600)
   res=serialset("IR",1)
   res=serialset("IR",32769) but not work with T with all palms it works
This code has also been suggested: 
   res=serialopen(32769,9600)
   res=serialset("PortID",sysinfo(3))
   res=serialset("IR",1)

To force serial comms on a TungstenT use port 32771 in the serialopen statement. RAW IR serial comms does not work on the TungstenT, Zire71, and other OS5 palms. This is known problem and can not be fixed.

2. Using Serial Comms on a device with USB port

To do serial communcations with a device that has a USB port, you'll need the proper cable. If you have the right cable, you should be able to open and use the serial port normally.
