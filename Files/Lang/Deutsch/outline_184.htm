SysTrapFunc                                                                                         Funktion

 

SYSTRAPFUNC(trapnum, numargs[, arg1[, arg2[, ...]]]))

 

Beschreibung

Trapnum ist eine Palm OS Konstante die bestimmt, welche Systemroutine gerufen werden soll. Numargs enthält die Anzahl der Parameter, die der Systemroutine übergeben werden müssen. Je nachdem welche Routine gerufen wird, können die weiteren Parameter variieren. Achten Sie besonders auf die Parameter und die Variablentypen wie auch den Ergebnistyp bei Systemroutinen. Die Tech Note 6 enthält komplette Informationen zu diesem Thema.

 

Beispiel

Res=SysTrapFunc(531, 4, 80, 0, 80, 160)  'ruft trap 0xA213 (WinDrawLine)
