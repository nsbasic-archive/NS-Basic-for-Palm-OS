AppLaunch                                                                                             Funktion

 

APPLAUNCH( cardNo as integer, pgm as string, cmd as integer, data as string)

 

Beschreibung

liefert eine Ganzzahl mit dem Antwortcode des Aufrufes. CardNo sollte auf 0 gesetzt werden, pgm ist ein String der den Namen der aufzurufenden Anwendung enthält, cmd enthält die Parameter als String, die an pgm übergeben werden sollen. Diese Funktion ruft eine andere Anwendung von NS Basic/Palm aus auf. Anders als bei CHAIN wird die aufrufende Anwendung nach dem Aufruf weiter ausgeführt.

 

Beispiel

' in dem code eines objektes:

PalmPrint "Hello World"  'ruft die PalmPrint function im Code auf

...

sub PalmPrint(PrintData as string)

   dim res as integer

   res=AppLaunch(0, "PalmPrint", 32768, PrintData)

end sub

'Dieses Beispiel ruft PalmPrint von Steven's Creek (http://www.stevenscreek.com) auf, um "Hello World" an den IR Drucker zu senden.
