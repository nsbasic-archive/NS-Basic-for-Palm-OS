Format                                                                                                      Funktion

 

FORMAT( numVar as anyNumType,  fmtstring as String)

 

liefert: Einen formatierten Textstring

 

Beschreibung

Dient dazu, numerische Daten in Strings (Zeichenfolgen) zu formatieren, um diese korrekt darzustellen. NumVar kann jeder numerische Datentyp (integer, short, float oder double) sein. FmtString ist ein String, der die Formatierung der Zeichenfolge angibt. Die Ergebnisvariable sollte genügend Zeichen enthalten, um das Ergebnis aufnehmen zu können. FmtString kann ein oder mehrere der folgenden Zeichen enthalten:

 

0

Wird durch eine Ziffer der eingegebenen Nummer ersetzt, oder durch 0, sofern keine Ziffer existiert

n

Wird durch eine Ziffer der eingegebenen Nummer ersetzt, oder durch ein Leerzeichen, sofern keine Ziffer existiert

#

Wird durch eine Ziffer der eingegebenen Nummer ersetzt, oder weggelassen, sofern keine Ziffer existiert

.

Platziert einen Dezimalpunkt in der Ausgabe

+

Wird durch ein Minuszeichen ersetzt, wenn die Zahl negativ ist, ansonsten durch ein Leerzeichen

-

Wird durch ein Pluszeichen ersetzt, wenn die Zahl positiv ist, ansonsten durch ein Leerzeichen

,

Setzt ein Komma in die Ausgabe wenn eine vorhergehende Ziffer vorhanden ist, ansonsten wird nichts dargestellt

 

Alle anderen Zeichen werden in die Ausgabe mit übernommen

 

Beispiel

Dim a as double

Dim b as double

Dim c as double

Dim i as integer

Dim j as integer

Dim k as Integer

Dim s as String

a=123.45

b=-37.65

c= 12345.6

i=92

s="         "

s=Format(a, "#####.000" ) 'liefert 123.450   (länge=7 da # keine Leerzeichen reserviert)

s=Format(a, "nnnnn.000" ) 'liefert   123.450 (länge=9 da n Leerzeichen reserviert)

 

s=Format(a, "-00000.000") 'liefert 00123.450 (Minuszeichen ignoriert, da negativ)

s=Format(b,"+nn,nnn.0") 'liefert -37.6  

s=Format(c, "nn,nnn.00") 'liefert 12,345.60

s=Format(i, "###%")    'liefert 92%
