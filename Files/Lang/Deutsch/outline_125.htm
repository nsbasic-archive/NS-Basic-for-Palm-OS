Methods                                                                                                       Objekt

 

Methoden ändern das Aussehen oder die Einstellungen eines Objektes. Jedes Objekt hat bestimmte Methoden, die genutzt werden können. Schlagen Sie in der Dokumentation eines Objektes nach, welche Methoden anwendbar sind.

 

Add Itemtext, indexNo[,NODISPLAY]

Füge ItemText der Liste an der Position indexNo hinzu.NODISPLAY verhindert die Neuzeichnung des Objektes. Für Grid-Objekte (Raster) gibt es keine Argumente: es wird lediglich eine neue Zeile am Ende der Tabelle angefügt.

Clear

Löscht ein Objekt. Wenn es sich beim Objekt um ein Form handelt, werden alle Felder gelöscht. („“). Wenn es sich um ein Raster handelt, werden alle Zeilen gelöscht.

Count

Gibt die Anzahl von Objekten eines Form- Objektes zurück.

Hide

Versteckt ein Objekt

ID

Liefere die interne ID Nummer des Objekts (Integer), hilfreich für API Aufrufe.

Index

Positionsnummer eines Objektes auf dem Formular. Beginnt mit 0. Wenn das Formular einen Titel hat, ist dies die Nummer 0.

ItemNo item[, index]

Füge item zur Liste an Position indexNo. Wenn indexNo 1 ist, füge am Anfang der Liste  ein. Wenn kein indexNo mitgeliefert wird, füge am Ende der Liste ein.

Itemtext IndexNo

Liest ein Element aus einer Liste

Max int

Liest oder Schreibt den maximalen Wert einer Scrollleiste

MaxChars int

Liest oder Schreibt die maximale Anzahl von Zeichen

Min int

Liest oder Schreibt den minimalen Wert einer Scrollleiste

NoItems

Liest die Anzahl Elemente einer Liste

PageSize int

Liest oder Schreibt die Seitengröße einer Scrollleiste

Redraw

Zeichnet ein Objekt neu

Remove indexNo

Entfernt das Element an der Position indexNo aus der Liste

Selected indexNo

Liest oder Schreibt die Auswahl

SetFocus

Setzt den Fokus auf ein Objekt

Show

Zeige das Objekt

Status

Wird von CheckBoxen und PushButtons verwendet

Text

Liest oder Schreibt den Text eines Labels

Value CurValue

Liest oder Schreibt den aktuellen Wert einer Scrollleiste

 

Beispiel

MyField.text="Reg Llama"   'setzt den Text eines Feldes

MyPopupList.add "Brixton" 'fügt den Namen einer Popupliste hinzu

MyCheckbox.setFocus        'setzt den Fokus auf das Objekt

Msgbox MyOtherField.text   'gibt den Wert eines Feldes aus

MyCheckbox.status=nsbChecked      'setzt eine Checkbox
