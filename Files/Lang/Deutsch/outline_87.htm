FillRectangle                                                                                               Befehl

 

FILLRECTANGLE x as integer, y as integer, width as integer, height as integer,

cornerDiam as Integer [, penType]

 

Beschreibung

Füllt ein Rechteck mit den angegebenen Koordinaten x, y und den Ausmaßen width, height, wobei CornerDiam den Radius der abgerundeten Ecken angibt. Ein Radius von 0 wird keine abgerundete Ecke erzeugen. Die Art des Stiftes penType ist optional – wenn Sie angegeben ist, muss dies entweder nsbNormal, nsbInverted, nsbGrey oder nsbCustom sein. Sehen Sie sich für weitere Informationen dazu das ‚Pattern’- Beispiel an.

 

Beispiel

FillRectangle 10, 15, 30, 40, 5, nsbGray
