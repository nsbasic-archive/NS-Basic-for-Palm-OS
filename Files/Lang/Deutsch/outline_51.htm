DbCreateDatabaseFromResource                                                   Funktion

 

DBCREATEDATABASEFROMRESOURCE(ResourceType as String, ResourceID as Integer)

 

Beschreibung

Diese Funktion erlaubt es Ihnen, Datenbanken und Shared Libraries, die in Ihrer prc Datei stehen, auf Ihrem PDA zu extrahieren und installieren, so als wären sie per HotSync transferiert worden.

 

Zum Hinzufügen einer Ressource zu Ihrem Projekt benutzen Sie die „Add Ressource“ Option des Projekt Menüs oder klicken Sie die rechte Maustaste im Projekt Explorer. Wählen Sie die Datei auf Ihrem PC aus, die Sie als Ressource hinzufügen wollen. Das können pdb oder prc Dateien sein. Die ResourceID wird vom System generiert. Der ResourceType ist standardmäßig auf DBIM eingestellt, kann aber auf jeden anderen Wert geändert werden. Die maximale Größe einer Ressource ist auf 64k festgelegt. Ihre prc oder pdb Datei darf also nicht größer sein.

 

Zur Laufzeit können Sie DbCreateDatabaseFromResource zu jeder Zeit aufrufen, um eine Shared Library oder Datenbank zu extrahieren. Die Funktion gibt einen Fehlercode zurück. Die Bedeutung der Fehlercodes können Sie der Tabelle zu DbOpen entnehmen.

 

Beispiel:

Dim res as integer

res=dmCreateDatabaseFromResource("DBIM",1015)
