TestNum                                                                                                  Funktion

 

TESTNUM(theString as string, SignOption as string, NoDigitsBeforeDecPt as integer, NoDigitsAfterDecPt as integer)

 

Beschreibung

Überprüft den Inhalt eine String Variabeln um zu sehen ob:

1. der String einen numerischen Inhalt hat.

2. Wenn signOption leer (“ “) ist, so darf der String keine Vorzeichen (+, -) haben, enthält signOption ein „+“ oder „-“, so sind die Vorzeichen erlaubt.

3. Die Anzahl Vorkommastellen im String darf NoDigitsBeforeDecPt nicht übersteigen.

4. Die Anzahl Nachkommastellen im String darf NoDigitsAfterDecPt nicht übersteigen.

 

Liefert 0 wenn alle Tests in Ordnung waren, ansonsten einen Wert ungleich 0.

 

Beispiel

Dim result as integer

Dim temp as string

 

Temp=fldVoltageLevel.text 'liest den String aus einem Feldobjekt

 

'erlaubt + oder – Zeichen, Vorkommastellen und 2 Nachkommastellen

If Not TestNum(temp, "-" , 3, 2)=0 then

  Msgbox "Der Wert wurde korrekt eingegeben"

End if
