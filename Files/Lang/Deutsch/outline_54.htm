DbFind                                                                                                     Funktion

 

DBFIND(dbName as Database, dbKey as anyVarType)

 

Beschreibung

Ermittelt, ob ein Datensatz mit dem angegebenen Schlüssel in der übergebenen Datenbank existiert. Die Datenbank muss bereits mittels dbOpen geöffnet sein. DbKey muss eine Variable sein, kein Ausdruck. Der Fehlercode wird bei dbOpen detailliert beschrieben.

 

Beispiel

Dim err as Integer

Err=DbFind(CustomerDb, “362”)

If err>0 then

  Msgbox “Konnte Kundendatensatz 362 nicht finden !”

end if
