Slider                                                                                                             Objekt

 

Ein SLIDER Objekt (Schieberegler) repräsentiert einen Wert innerhalb eines defienierten Bereiches. Eine Slider kann z.B. einen Wert zwischen 0 und 10 darstellen. PageSize bezeichnet den Wert um den Value sich ändert, wenn der Slider um einen Schritt bewegt wird.

 

unterstütze Eigenschaften (während der Entwicklungsphase setzen)

Maximum, Minimum, Page Size, Value, Visible

 

unterstütze Methoden (siehe „Methoden“)

Left, Top, Width, Height, Hide, Show, Redraw, ID, Index

 

Beispiel

Record=mySlider.Value / mySlider.Width * dbGetNoRecs(myDatabase)
