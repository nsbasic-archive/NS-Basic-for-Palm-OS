Label                                                                                                             Befehl

 

labelName:

 

Beschreibung

Definiert ein Label (Sprungmarke) innerhalb einer Funktion oder Prozedur, an die mittels GoTo oder Gosub gesprungen werden kann.

 

Beispiel

if loanType=“home” then

  GoTo homeLoanCalculations

Else

  GoTo carLoanCalculations

End if
