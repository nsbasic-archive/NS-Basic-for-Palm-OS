Selector                                                                                                        Objekt

 

Ein SELECTOR Objekt stellt einen Text innerhalb eines grauen Rechtecks dar. Normalerweise wird dies benutzt, um Datum oder Zeit darzustellen. Wenn der Anwender darauf klickt, veranlasst der mit dem Objekt verbundene Code einen Datums- oder Zeitpopup, damit der Anwender ein neues Datum oder eine neue Zeit eingeben kann. Die Breite des Objektes wird während der Laufzeit dynamisch anhand des Textes und des eingestellten Zeichensatzes ermittelt.

 

unterstütze Eigenschaften (Set at design time)

Label, Font ID, Anchor Left, Visible

 

unterstütze Methoden (siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Text, ID, Index, Type
