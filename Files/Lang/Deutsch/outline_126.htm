Mid                                                                                                            Funktion

MID(theString as String, start as Integer, num as Integer)

 

Beschreibung

Liefert einen Teilstring des Strings theString, beginnend von start mit der Länge von num Zeichen. Sollte start außerhalb der Länge des Strings sein, so wird ein Null-String zurückgeliefert. Wenn start + num länger als der String ist, so wird das Ergebnis auf die Stringlänge verkürzt. Dabei der rechte Teil des Strings, beginnend bei Position start zurückgeliefert.

 

Beispiel

Dim result as String

Result=Mid(“John Doe”, 3, 4) ‘liefert result=“hn D”
