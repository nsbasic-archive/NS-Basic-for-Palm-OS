SerialReceiveWithEvent                                                                      Funktion

 

SERIALRECEIVEWITHEVENT()

 

Beschreibung

Diese Funktion bringt den PDA dazu, auf eingehende Daten der seriellen Schnittstelle zu warten. Sobald Zeichen empfangen werden, wird ein NsbSerialEvent ausgelöst und Sie können die Daten durch Ihre Eventcode Routine auswerten lassen. Die Funktion benötigt keine Parameter. Die Ergebnis Codes sind unter SerialOpen beschrieben. Erfordert Palm OS 3.5 oder höher und wird nicht von allen Versionen von Palm OS 5 unterstützt. Als Workaround für Geräte, die diese Funktion nicht unterstützen, empfiehlt es sich, einen Timer zum regelmäßigen Kontrollieren des Serial Ports zu verwenden.

 

Beispiel

'anywhere in your program after serial port is opened

Global err as integer

err=SerialReceiveWithEvent()

 

'in your form's Event Script

   

    Dim received as String

    If getEventType()=nsbSerialIn Then

        MsgBox "Serial In Event"

        err=serialReceive(received, 10, 0)

      If Err > 0 Then

        Beep

        error.text=str(err)

        data.text=data.text+"."+Received

      Else

        error.text=""

        data.text=data.text+received

      End If

    End If
