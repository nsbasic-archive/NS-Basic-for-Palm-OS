Type                                                                                                              Befehl

 

TYPE userName

 

Beschreibung

Beginn einer Definition eines Benutzerspezifischen Variablentyps oder einer Struktur.

 

Beispiel

Type PersonType

  Name as string

  Occupation as string

  Salary as double

End Type

 

Dim person as PersonType

Person.name="John Doe"

Person.Occupation="Freelance artist"

Person.Salary=50000.00

msgbox "Der Name ist " + person.name

msgbox "Der Beruf ist " + person.occupation

msgbox "Das Gehalt ist " + str(person.salary)

 

Es können auch Arrays mit benutzerdefinierten Typen angelegt werden:

Dim workers(100) as PersonType

Sum=0

For I=1 to noWorkers

  Sum=Sum + workers.salary(i)

Next

Avg=sum / noWorkers
