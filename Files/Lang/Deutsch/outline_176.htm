Str                                                                                                              Funktion

 

STR(theNumber as Float)

 

Beschreibung

Liefert einen String, indem theNumber in eine interne Fließkommazahl interpretiert, und anschließend in eine Zeichenfolge umgewandelt wird. Negative Zahlen erhalten ein führendes Minuszeichen. (z.B. „-12.5“)

 

Beispiel

Dim result as string

Dim x as Float

X=-22.56

Result=Str(x) ‘liefert result=“-22.56”
