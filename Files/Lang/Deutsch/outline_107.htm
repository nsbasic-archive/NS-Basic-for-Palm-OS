InStr                                                                                                          Funktion

 

INSTR(start as integer, s as string, pattern as string, type as integer)

 

Beschreibung

Durchsucht den String s von der Startposition start um die in pattern angegebene Zeichenfolge zu finden. Wenn der Typ type 0 ist, wird nach einem exakten Vergleich gesucht. Bei Typ=1 wird nicht nach der Groß- und Kleinschreibung unterschieden. Eine Ganzzahl wird zurückgeliefert, die entweder 0 ist, sofern pattern nicht gefunden wurde, ansonsten die Startposition der gefundenen Zeichenfolge in s.

 

Beispiel

Dim s as string

Dim result as integer

s="abcdef"

result=InStr(1, s, "cde", 0) ‘liefert 3

result=InStr(1, s, "CDE", 0) 'liefert 0 wg. Groß/Kleinschreibung

result=InStr(1, s, "CDE", 1) 'liefert 3
