Alert                                                                                                          Funktion

 

ALERT(title as String, msg as String, type as Integer, button0 as String, button1 as String, ... , ButtonN as String)

 

Beschreibung

Zeigt eine Palm OS Nachrichtenbox des Typs “type“ an, und wartet darauf, dass der Benutzer eine der Schaltflächen aktiviert um die Nummer der aktivierten Schaltfläche zurück zu liefern. Die Zählung startet dabei bei 0.

 

Type beschreibt den formatierten Typ des Nachrichtenfensters. title ist ein String, der als Kopf des Fensters angezeigt wird.

Msg ist ein String, der den Inhalt der darzustellenden Nachricht enthält.. Button, button1, usw. sind die Bezeichnungen der einzelnen Schaltflächen, die dargestellt werden sollen. Sie können generelle so viele Schaltflächen verwenden wie Sie möchten, im Allgemeinen aber werden nur 2 bis 3 Schaltflächen verwendet..

 

Type=0

zeige ein Nachrichtenfenster des Typs ‘Information’ an

1

zeige ein Nachrichtenfenster des Typs ‘Bestätigung’ an

2

zeige ein Nachrichtenfenster des Typs ‘Warnung’ an

3

zeige ein Nachrichtenfenster des Typs ‘Error’ an

 

Beispiel

Dim result as Integer

Result=Alert("Get metric decision.......", "Do you want to use meters or feet?",0,"Meters","Feet")

If result=0 then

  ..................

else

  ..........................

end if
