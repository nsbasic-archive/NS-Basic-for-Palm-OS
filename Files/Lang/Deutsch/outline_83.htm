Exit Funktion                                                                                               Befehl

 

EXIT FUNCTION

 

Beschreibung

Beendet die Ausführung einer Funktion vor ihren eigentlichen definierten Ende mittels End Function, und setzt die Programmausführung an der Stelle fort, von der aus die Funktion aufgerufen wurde.

 

Beispiel

Function a(count as integer) as Float

  If count > 100 then

    A=0

    Exit Funktion

  End if

  A=Count * 1.5

End Function
