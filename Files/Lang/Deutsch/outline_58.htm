DbOpen                                                                                                   Funktion

 

DBOPEN(dbName as Database, fileName as String, cardNo as Integer)

 

Beschreibung

Öffnet eine bestehende Datenbank auf der angegebenen Speicherkarte des Palm OS Gerätes. Liefert einen ganzzahligen Fehlercode zurück.

 

DbName ist die Referenz auf die Datenbank innerhalb des Programms. Alle Datenbankaktionen werden durch diesen Namen identifiziert. FileName beinhaltet den Dateinamen, mit dem die Datenbank im Palm OS Speicher registriert wurde. CardNo bezeichnet die Speicherkarte, auf der die Datenbank existiert. Normalerweise ist dies 0. Ein Datenbankname muss durch DIM dbname as Database definiert werden, bevor er durch die dbOpen Funktion verwendet werden kann.

 

Ein Aufruf von dbOpen ist Grundvoraussetzung für alle weiteren Datenbankaufrufe.
