Label                                                                                                             Objekt

 

LABEL Objekte werden verwendet, um Texte auf einem Formular anzuzeigen. Labels werden oft dazu herangezogen um Titel, Anweisungen oder Bezeichnungen anderen Objekten voranzustellen. Der Text eines Labels kann durch das Programm geändert werden, aber er kann nicht länger gemacht werden, als dies während der Entwicklungsphase eingegeben wurde. Füllen Sie das Label notfalls mit Leerzeichen auf, um den nötigen Platz zu reservieren.

 

unterstützte Eigenschaften(während der Entwicklungsphase setzen)

Left, Top, Font ID, Label

 

unterstützte Methoden(siehe "Methoden")

Hide, Show, Redraw, text, ID, Index, Type

 

Beispiel

MyLabel.text="Enter Name:"

MyLabel.redraw
