PushButton                                                                                                 Objekt

Ein PUSHBUTTON ist ein Bereich der einen Text enthält und wahlweise “nicht selektiert”, also normal dargestellt, oder “selektiert”, also invertiert dargestellt, sein kann. Der Anwender ändert einen Pushbutton von „selektiert“ nach „nicht selektiert“ (oder andersherum) indem er einfach darauf klickt. Pushbutton Objekte werden normalerweise verwendet, um eine mögliche Auswahl von Parametern darzustellen, wobei mögliche Ergebnisse Ja/Nein, An/Aus, usw. sind. Verwenden Sie die GroupID um Pushbuttons exklusiv zu machen.

 

unterstützte Eigenschaften(während der Entwicklungsphase setzen)

Label, Font ID, Text, Group ID, Anchor Left, Selected, Visible

 

unterstützte Methoden(siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Text, Status, ID, Index, Type

 

Beispiel

Msgbox "The PushButton status is " + str(selYesNo.status)

if mydb_yes_no_fld=1 then selYesNo.status = nsbOn else selYesNo.status = nsbOff
