Gosub                                                                                                           Befehl

 

GOSUB labelName

 

Beschreibung

Transferiert die Programmausführung an einen durch das Label labelName bezeichnete Stelle innerhalb der aktuellen Funktion oder Prozedur. Von dort aus kann die Ausführung wieder mittels des RETURN Ausdruckes zurückgegeben werden.

 

Beispiel

 

Gosub routine1

...

routine1:

  ....

  ....

Return
