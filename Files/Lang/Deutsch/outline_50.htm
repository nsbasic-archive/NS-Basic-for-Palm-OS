DbCreate                                                                                                 Funktion

 

DBCREATE(dbName as Database, fileName as String, cardNo as Integer, creator as String)

 

Beschreibung

Erzeugt eine neue Datenbank auf der Speicherkarte cardNo des Palm OS Gerätes (normalerweise 0), und liefert einen ganzzahligen Fehlercode zurück, der bei dbOpen detailliert beschrieben wird.

 

DbName referenziert den Datenbanknamen innerhalb des Programms. Alle Aktionen mit dieser Datenbank beziehen sich auf diesen Namen. FileName bezeichnet den Namen der Datenbankdatei, mit dem diese im Palm OS Speicher abgelegt wird. CardNo ist auf allen modernen Geräten 0.Creator ist ein 4-Zeichen String, der als CreatorID fungiert, um den Eigentümer der Datenbank zu identifizieren. Im Normalfall sollten Sie die gleiche Creator ID verwenden, die Sie für Ihr Projekt definiert haben. Um Dubletten zu vermeiden, erlaubt Palm Computing Ihnen eigene CreatorIDs zu reservieren. Dies können Sie auf der Palm Computing Webseite kostenlos tun, so dass gewährleistet ist, dass die CreatorIDs eindeutig bleiben. Die Datei wird mit dem gesetzten Backup Attribut erzeugt, was zu Folge hat, dass HotSync diese automatisch auf Ihrem Desktop speichert. Die Palm Namenskonvention setzt sich zusammen aus Ihrem Datenbanknamen, einen Unterstrich gefolgt von Ihrer CreatorID.

 

Beispiel

Dim result as integer

Dim CustomerDb as database

Result=DbCreate(CustomerDb, “Company-Test”, 0, “JimB”)
