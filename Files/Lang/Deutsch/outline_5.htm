05.E Sonstige Befehle

 

Beep

erzeugt einen hörbaren Ton

Chain

beendet die aktuelle Anwendung, und startet eine andere PalmOS Anwendung

Cursor

bewegt den Zeiger zu den angegebenen Koordinaten

Define

definiert eine Funktion vor ihrem eigentlichen Code, so dass andere Funktionen und Prozeduren korrekt darauf zugreifen können.

Delay

verzögert die Programmausführung für die angegebene Dauer, und setzt die Anwendung in den Warte-Status, bis die Warteperiode abgelaufen ist.

Dim

definiert Variabeln, Listen und Datenbanken die im Code verwendet werden

Display

zeigt den Inhalt einer oder mehrerer Variablen an der Zeigerposition an

Global

definiert Variabeln die global im gesamten Code aller Funktionen und Prozeduren verwendet werden können.

MenuDraw

zeichnet das angegebene Menü auf den oberen Bereich der Anzeige

MenuErase

löscht das momentan angezeigte Menü von der grafischen Anzeige

MenuReset

bricht jedes ausgeklappte Untermenü ab, das bereits selektiert wurde

Msgbox

zeigt ein Nachrichtenfenster auf der Anzeige an, die keine Antwort des Benutzers erlaubt. Ansonsten muss die Funktion ALERT eingesetzt werden

SetEventHandled

teilt dem Betriebssystem mit, das Ereignis nicht zu beachten

Sound

erzeugt einen Ton mit der angegebenen Frequenz und Dauer
