DbGet                                                                                                       Funktion

 

DBGET(dbName as Database, aVariable as varList)

 

Beschreibung

Liest eine Variablenliste aus dem aktuellen Datensatz an dem momentanen Offset. Die Variablen in aVariable kann jeden gültigen Typ (String, Fließkomma, Ganzzahl, usw.) haben. Die Datenbank muss bereits mittels dbOpen geöffnet  worden sein. Am Ende der Datenbank wird –1 zurückgeliefert, ansonsten entspricht der Fehlercode dem von dbOpen.

 

Beispiel

Dim res as Integer

Res=DbGet(OrdersDb, OrderNo)
