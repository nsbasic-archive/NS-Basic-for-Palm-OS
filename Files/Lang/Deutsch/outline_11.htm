06.E integrierte Zeitfunktionen

 

Hour

liefert die Stundenzahl einer Eingabezeit

HourMin

liefert einen String im Format “HH:MM” basierend auf der Eingabezeit

HourMinAMPM

liefert einen String im Format “HH:MM XX” wobei XX für AM oder PM steht

Minute

liefert die Minutenzahl einer Eingabezeit

Now

liefert die aktuelle Uhrzeit, basierend auf der Systemzeit

PopUpTime

fragt den Benutzer nach einer Zeit, unter Verwendung des Standard Dialogs

Second

liefert die Sekundenzahl einer Eingabezeit

TimeDiff

liefert die Anzahl an Sekunden zwischen zwei Zeitangaben zurück

TimeVal

liefert eine Zeit basierend auf der angegebenen Anzahl Stunden, Minuten und Sekunden

ToTime

liefert eine Zeit, basierend auf einem String im Format “HH:MM:SS”
