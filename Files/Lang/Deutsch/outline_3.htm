05.C Moduldefinitionen und Kontrollbefehle

 

Sub

definiert den Beginn einer Prozedur

Function

definiert den Beginn einer Funktion

End Sub

definiert das Ende einer Prozedur

End Function

definiert das Ende einer Funktion

Call

übergibt die Kontrolle an eine andere Prozedur

Exit Sub

verlässt die aktuelle Prozedur

Exit Function

verlässt die aktuelle Funktion
