05.B Kontroll- und Logik Befehle

 

If <test> Then

Else

ElseIf

End if

testet einen Ausdruck und führt Befehle in Abhängigkeit des Testergebnisses aus

Do

Loop

Exit Do

wiederholt eine Serie von Befehlen bis der Test die Schleife beendet.

For

Next

Exit For

wiederholt die Ausführung einer Serie von Befehlen und erhöht einen Zähler, bis der Test die Schleife beendet.

GoTo

überträgt die Kontrolle an eine Sektion, die durch das Label gekennzeichnet ist.

Gosub

Return

überträgt die Kontrolle an eine Sektion, die durch das Label gekennzeichnet ist, und springt anschließend wieder zurück.

NextForm

ändert das angezeigt Formular auf ein anderes Formular

Select Case

   Case

End Select

testet den Wert einer Variabeln und führt eine Serie von Befehlen aus, in Abhängigkeit des jeweiligen Wertes.

Stop

stoppt die Programmausführung
