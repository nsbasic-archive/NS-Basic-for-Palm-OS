Form                                                                                                              Objekt

Eine FORM (Formular) stellt einen Spezialfall eines Objekts dar. Es wirkt als Container für andere Objekte. Immer nur ein Form kann gerade aktiv sein. Mittels ‚NextForm’ können Sie von einem Form zum anderen wechseln. Wenn ein Form angezeigt wird, werden seine ‘Form Before’ und ‚Form After’ Skripte ausgeführt. Ein auf dem Form ausgeführtes Ereignis wird dem ‚Event Code’ des Formulars übergeben.

 

Unterstützte Eigenschaften (in der Entwicklungsphase setzen)

Name, ID, Default Form, Height, Left, Modal Form, Show TitleBar, Title, Top, Width

 

Unterstützte Methoden (Siehe "Methoden")

Clear, Count

 

Beispiel

Form1003.clear

Msgbox “The number of objects on this form is” + str(Form1003,count)
