Gadget                                                                                                          Objekt

 

Ein GADGET ist ein rechteckiger Bereich auf dem Formular, welcher wie eine Schaltfläche agiert, wobei Sie dafür verantwortlich sind, den jeweiligen Text in diesen Bereich zu schreiben oder zeichnen. Ein gadget hat kein Text oder Bild, welches dargestellt wird. Sobald der Anwender darauf klickt, wird der entsprechende Code der damit verbunden ist, ausgeführt.

 

Gadgets können auch dafür benutzt werden, um Unterschriften oder Zeichnungen als Bitmaps einzulesen. Sektion 10.I beschreibt dies näher, ebenso die Beispiele zum Einlesen von Unterschriften.

 

unterstützte Eigenschaften(währen der Entwicklungsphase setzen)

Left, Top, Width, Height, Visible

 

unterstützte Methoden(siehe "Methoden")

Hide, Show, Redraw, StartSignatureCapture, EraseSignature, DisplaySignature, EndSignatureCapture, ID, Index, Type
