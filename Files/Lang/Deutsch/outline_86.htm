Field                                                                                                               Objekt

 

Ein Field Objekt ist ein Bereich auf dem Formular, in dem der Anwender Text eingeben kann. Der Inhalt kann ebenfalls durch die Anwendung gesetzt werden. Wenn Sie einen Zeilenumbruch innerhalb des Textes erzwingen möchten, so verwenden Sie das Zeichen %h0A (Chr(10)).

 

Der Code des Feldes wird ausgeführt, sobald der Anwender das Feld verlässt indem er ein anderes Objekt oder Menü anklickt. Der Code kann dann Eingabeprüfungen oder, basierend auf dem Feldinhalt, weitere Berechnungen durchführen. Er wird ausgeführt, unabhängig davon ob sich der Feldinhalt geändert hat oder nicht. NS Basic/Palm speichert den vorherigen und neuen Wert intern nicht ab – es ist nur bekannt, dass der Anwender das Feld selektiert hat, und sich dann zu einem anderen Objekt bewegt hat.

 

Ein normales Programm liest Datensätze ein und stellt deren Inhalt mittels Felder für den Anwender sichtbar dar. Nun kann der Anwender Änderungen vornehmen, so dass das Programm diese Änderungen speichern kann. Dies bedeutet normalerweise, dass in dem Startcode des Formulars der Datensatz gelesen, und die Feldinhalte in jedes Feld geschrieben werden.

 

unterstützt Eigenschaften (in der Entwicklungsphase setzten)

Label, Font ID, Text, Max Characters, Left Justified, Underline, Single Line, AutoShift, HasScrollbar, Dynamic Size, Editable, Numeric, Visible

 

unterstützt Methoden (siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Text, Status, SetFocus, ID, Index, Type

 

Beispiel

' liest Datensätze ein, stellt sie dar und speichert die Änderungen.

Dim Name as string

Dim Age as integer

Dim Married as integer  '0=ledig oder 1=verheiratet

Dim err as integer

Err=DbRead(MyDb, Name, Age, Married) 'verwendet Name als Schlüssel

FldName.text=Name

FldAge.text=str(Age)

ChkMarried.status=Married

 

Die “Done” Schaltfläche die der Anwender betätigt nachdem er die Werte entsprechend modifiziert hat, könnte folgenden Code enthalten:

 

Dim Name as string

Dim Age as integer

Dim Married as integer

Dim err as Integer

If not testNum(fldAge.Age, " ",3,0)=0 then

  Beep

  Msgbox "Das eingegebene Alter ist ungültig."

  Exit sub

end if

Name=fldname.text

Age=val(fldAge.text)

Married=chkMarried.status

Err=DbUpdate(MyDb, Name, Age, Married)
