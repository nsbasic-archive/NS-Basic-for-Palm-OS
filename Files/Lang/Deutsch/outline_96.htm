GetKey                                                                                                     Funktion

 

GETKEY()

 

Liefert einen 1-Zeichen String, mit dem zuletzt gedrückten Zeichen oder Schaltfläche. Bestimmte Ereignisse erzeugen ebenfalls GetKey Sequenzen.

 

Beschreibung

Liefert die zuletzt gedrückte Taste oder Schaltfläche als 1-Zeichen String. Siehe die nachfolgenden Beispiele für die gelieferten Werte. Schauen Sie sich auch das Beispiel SpecEvent.prj im Beispielordner an. Diese Beispiel zeigt auch, wie die Palm 5 Way Tasten erkannt werden können. Um sich die reinen Zeichencodes, die ein Event liefert, anzusehen, können Sie die Funktionen SysInfo(9), SysInfo(10) und SysInfo(11) benutzen.

 

Beispiel

Im EventCode eines Formulares, verwenden Sie folgendes:

 

dim key as string

If Not GetEventType()=NSBKeyOrButton then Exit Sub

key=getKey()

select case asc(key)

case 1     'fester Button #1 auf dem Device

...

case 2     'fester Button #2 auf dem Device

...

case 3     'fester Button #3 auf dem Device

...

case 4     'fester Button #4 auf dem Device

...

case 5     'Menü-Taste auf dem Silkscreenbereich

...

case 6     'Kommando in dem Grafittibereich

...

case 17    'Anwendungsstarttaste auf dem Silkscreenbereich

...

case 7     'Finden-Taste auf dem Silkscreenbereich

...

case 16    'Rechner-Taste auf dem Silkscreenbereich

...

case 11    'Scroll Hoch Schaltfläche

...

case 12    'Scroll Runter Schaltfläche

...

case 14    'An/Aus Knopf

...

case 15    'Cradle HotSync Knopf

...

case 18    ‘auto power off

...

 

case else

...

end select
