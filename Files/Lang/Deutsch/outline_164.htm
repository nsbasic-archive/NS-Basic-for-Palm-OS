SerialSend                                                                                              Funktion

 

SERIALSEND(buffer as string or integer array, noChars as integer)

 

Beschreibung

Überträgt Daten über den seriellen Port. Buffer enthält die zu sendenden Daten. Sofern Buffer ein Array aus Ganzzahlen ist, wird jedes Element als ASCII Wert interpretiert und gesendet (auch 0 Werte). NoChars enthält die Anzahl zu sendender Zeichen aus buffer. Der ganzzahlige Fehlercode wird bei SerialOpen näher beschrieben. Zum Löschen des Empfangspuffers, verwenden Sie SerialSend(„“,0).

 

Beispiel

Err=SerialSend(outString, Len(outString))

Err=SerialSend(chr(0),1) 'send a null character
