LoadLibrary                                                                                                 Befehl

 

LOADLIBRARY filename as string [,refname as string]

 

Beschreibung

LoadLibrary erzeugt ein Code Objekt, welches dann verwendet werden kann, um externe Methoden innerhalb der gemeinsam genutzten Bibliothek aufzurufen. Die Bibliothek filename wird als filename, oder, sofern angegeben, als refname geöffnet. Siehe Tech. Note 5 für weitergehende Informationen über den Gebrauch der gemeinsam genutzten Bibliotheken.

 

Beispiel

Dim username as String

LoadLibrary "NSBSystemLib",“NSL“    'Verwendet NSBSystemLib.inf, mit refname = NSL

NSL.SyncUserName()               ‚ermittelt die HotSync ID
