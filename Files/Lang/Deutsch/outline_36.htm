CheckBox                                                                                                    Objekt

Ein CHECKBOX Objekt ist ein Textfeld, mit einer vorangestellten quadratischen box, die wahlweise markiert oder nicht markiert sein kann. Sobald der Benutzer auf die Box klickt, markiert er diese entweder als „checked“ oder als „not checked“, je nachdem welchen Wert sie vorher besessen hat. CheckBoxen werden normalerweise eingesetzt, um dem Benutzer eine Auswahl bestimmter Parameter oder Werte anzubieten, wobei die möglichen Antworten ja/nein, ein/aus, usw. sind. Verwenden Sie die Group ID Eigenschaft, um die Checkbox exklusiv zu gestalten.

 

unterstützte Eigenschaften (setzen während der Entwicklungsphase)

Label, Font ID, Text, Group ID, Anchor Left, Selected, Visible

 

unterstützte Methoden (siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Text, Status, ID, Index, Type

 

Beispiel

Msgbox "Der Status der checkbox ist " + str(chkYesNo.status)

If mydb_yes_no_fld=1 then chkYesNo.status = nsbChecked else chkYesNo.status = nsbUnchecked
