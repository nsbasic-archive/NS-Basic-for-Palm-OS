DrawRectangle                                                                                           Befehl

 

DRAWRECTANGLE xStart as Integer, yStart as Integer, width as Integer, height as Integer, cornerDiam as Integer [, penType]

 

Beschreibung

Zeichnet ein Rechteck von den angegebenen Koordinaten xStart, zStart mit der jeweiligen Breite und Höhe width und height. Durch CornerDiam können Sie den Radius der abgerundeten Ecken bestimmten – ein Radius von 0 wird keine abgerundete Ecke erzeugen. Die Art des Stiftes penType ist optional – wenn Sie angegeben ist, muss dies entweder nsbNormal, nsbInverted, oder nsbGrey sein.

 

Beispiel

DrawRectangle 10,10,30,40,0

DrawRectangle 40,50, 10,15,3,nsbInverted
