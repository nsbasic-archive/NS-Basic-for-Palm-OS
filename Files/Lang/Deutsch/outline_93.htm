Function                                                                                                       Befehl

 

FUNCTION functionName(varList) AS resType

 

Beschreibung

Definiert den Beginn einer Funktion, die übergebenen Variablen als Parameter, und den Typ des Ergebnisses. Eine Funktion gibt immer einen Wert an das aufrufende Programm zurück. Jede Variable in varList ist definiert als VariablenName AS varType, wobei varTyp der Typ der Variabeln (Integer, Float,usw.) ist. Variabeln müssen durch Komma getrennt werden. Sie können von jedem gültigen Variablentyp sein (Integer, Float, Array, UDT, etc.), außer dem Typ Database.

 

Ein Function Ausdruck definiert den Beginn einer Funktion, die durch den End Function Ausdruck beendet wird. Das Ergebnis einer Funktion wird zugewiesen, indem irgendwo innerhalb der Funktion dem Funktionsnamen der entsprechende Wert zugewiesen wird. Array und UDT Datentypen können nicht zurückgegeben werden.

 

Bei dem Erreichen des Ausdruckes End Function wird das Programm an der aufrufenden Stelle weiter ausgeführt. Sollten keine Parameter für die Funktion verwendet werden, so müssen die Klammern leer gesetzt werden „()“.

 

Beispiel

Funktion Calc1(rate as Float, loanAmt as Float) as Float

  Calc1=rate * loanAmt     'dieser Wert wird zurückgegeben

End Funktion
