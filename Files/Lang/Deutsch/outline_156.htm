Select Case / Case / End Select                                                             Befehl

 

SELECT CASE varname

    CASE value1

        Commands Group 1

    CASE value2

        Commands Group 2

…

    CASE ELSE

        Commands Group n

END SELECT

 

Beschreibung

Dient dazu, Befehle und Kommandos in Abhängigkeit des Wertes von varName auszuführen, wobei nach mehreren Bedingungen geprüft wird. Der CASE ELSE ist optional und wird verwendet, wenn keine der vorherigen Bedingungen zugetroffen hat. Die Werte für CASE Statements müssen Variablen oder Literale sein. Ausdrücke sind nicht erlaubt.

 

Beispiel

Dim PolicyType as String

.....

Select Case PolicyType

  Case "Auto"

    CalcAutoPremium

    ShowAutoPolicy

  Case "Home"

    CalcHomePremium

    ShowHomePolicy

  Case Else

    CalcGenericPremium

    ShowGenericPolicy

End Select
