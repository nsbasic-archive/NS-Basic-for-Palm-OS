Button                                                                                                           Objekt

Eine Button ist eine rechteckige Box mit abgerundeten Ecken, welche im Innenraum einen Text beinhaltet. Buttons (Schaltflächen) werden normalerweise dazu benutzt, mögliche Aktionen anzuzeigen. Wenn der Benutzer auf einen Button klickt, wird der dazugehörige Code ausgeführt.

 

unterstützte Eigenschaften (Setzen während der Entwicklungsphase)

Left, Top, Width, Height, Label, Font ID, Anchor Left, Frame, Non-bold Frame, Visible

 

unterstützte Methoden (siehe "Methoden")

Hide, Show, Redraw, Label, ID, Index, Type
