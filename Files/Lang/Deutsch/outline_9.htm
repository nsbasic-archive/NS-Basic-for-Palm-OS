06.C Integrierte String Funktionen

 

Asc

liefert den ASCII Wert (0 bis 255) des linken Zeichens des Arguments

Chr

liefert einen Ein-Zeichen String, dessen ASCII Wert angegeben ist

Format

formatiert einen numerischen Wert in einen String

InStr

prüfe, ob ein String in einem anderen String enthalten ist

LCase

liefert einen String aus Kleinbuchstaben, basierend auf dem Eingabestring

Left

liefert die linksbündig angegebene Anzahl Zeichen eines Strings

LeftPad

liefert die linksbündig angegebene Anzahl Zeichen eines Strings, aufgefüllt mit Leerzeichen

Len

liefert die Länge eines Strings

Ltrim

entfernt führende Leerzeichen eines Strings

Mid

liefert einen definierten Bereich eines Strings

Proper

liefert einen String, in dem jedes erste Zeichen großgeschrieben ist

Right

liefert die rechtsbündige angegebene Anzahl Zeichen eines Strings

RightPad

liefert die rechtsbündige angegebene Anzahl Zeichen eines Strings, aufgefüllt mit Leerzeichen

Rtrim

entfernt anhängende Leerzeichen eines Strings

Str

liefert den Wert eines numerischen Ausdruckes als String

TestNum

prüfe, ob ein String einen rein numerischen Inhalt hat

Trim

entfernt führende und anhängende Leerzeichen

Ucase

liefert einen String aus Großbuchstaben, basierend auf dem Eingabestring

Val

liefert den numerischen Inhalt eines Strings
