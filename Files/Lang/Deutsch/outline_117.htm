List                                                                                                                 Objekt

Ein LIST Objekt ist ein Rechteck auf dem Formular in dem mehrere Strings (Elemente) dargestellt werden können, jedes Element in einer separaten Zeile. Ein Element der Liste kann markiert (highlighted) werden, und wird invertiert dargestellt, mit weißen Buchstaben auf schwarzem Hintergrund. Die Rechteckige Bereich ist nur geeignet, eine wenige Elemente anzuzeigen. Wenn es mehr Elemente als Raum zum Darstellen gibt, erscheint eine Scrollbar auf der rechten Seite des Bereiches, um hoch- oder runter zu scrollen. Die Selected Methode wird gesetzt, bevor der Code der Liste ausgeführt wird, während der Text danach gesetzt wird.

 

Listeninhalte können während der Designphase erstellt werden, indem die Listbox ausgefüllt wird, die erscheint, wenn die Listeneigenschaft angeklickt wird. Die Inhalte der Liste kann auch während der Programmausführung mittels der Add Methode gesetzt werden. Die beste Stelle um dies zu tun ist normalerweise der After Code des Formulars, auf dem das LIST Objekt platziert ist. Verwenden Sie NODISPLAY um zu verhindern, dass das Formular neu gezeichnet wird, während Sie Elemente hinzufügen. Um kein Element auszuwählen, setzen Sie .Selected auf 0. Die Hide und Show Methoden funktionieren erst ab Palm OS 3.5 korrekt.

 

unterstützte Eigenschaften (während der Entwicklungsphase setzen)

Font ID, Visible Items, Visible, List

 

unterstützte Methoden (siehe "Methoden")

Left, Top, Width, Hide, Show, Redraw, Add, Remove, Clear, Text, Selected, NoItems, ID, Index, Type

 

Beispiel

Dim ItemsOnList as Integer

Dim selectedname as String

ListCity.clear 'die Liste immer zuerst löschen

ListCity.add "New York",NODISPLAY'Formular-Update verzögern

ListCity.add "Boston",NODISPLAY   'Formular-Update verzögern

ListCity.add "New Orleans"        'Formular-Updaten

ListCity.selected=2 'Boston markieren

ListCity.remove 3 'New Orlean löschen

ItemsOnList=ListCity.NoItems 'lese Anzahl der Elemente der Liste

selectedName= ListCity.text(ListCity.selected)

ListCity.clear
