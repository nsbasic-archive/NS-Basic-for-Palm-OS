DbPosition                                                                                              Funktion

 

DBPOSITION(dbName as Database, dbRecNo as Double, dbOffset as Integer)

 

Beschreibung

Positioniert die Datenbank auf die angegebene Datensatznummer dbRecNo und dort an den Offset innerhalb des Datensatzes dbOffset. Das nächste dbGet oder dbPut wird an diese Position Daten lesen/schreiben. Wenn dbRecNo größer als die Anzahl der Datensätze innerhalb der Datenbank ist, so wird die Datenbank automatisch vergrößert, so dass dbRecNo Datensätze vorhanden sind. Die Datenbank muss bereits mittels dbOpen geöffnet worden sein. Der Fehlercode wird bei dbOpen detailliert beschrieben.

 

Beispiel

Dim res as integer

Res=DbPosition(OrderDb, 15, 20) ‘gehe zu Datensatz 15, Offset 20.
