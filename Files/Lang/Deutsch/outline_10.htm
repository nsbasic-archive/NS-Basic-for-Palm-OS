06.D integrierte Datumsfunktionen

 

AddDays

addiert die angegebene Anzahl Tage zu dem Eingabedatum

AddMonths

addiert die angegebene Anzahl Monate zu dem Eingabedatum

AddYears

addiert die angegebene Anzahl Jahre zu dem Eingabedatum

DateDiff

liefert die Anzahl von Tagen zwischen zwei Datumsangaben zurück

DateVal

liefert einen Datumswert, basierend auf der angegebene Anzahl Tage, Monate, Jahre

DateMMDDYY

liefert einen String im Format “MM/DD/YYYY” basierend auf dem Eingabedatum

Day

liefert die Tageszahl eines Eingabedatums

DayOfWeek

liefert den Wochentag (1-7) eines Eingabedatums

DayOfYear

liefert den Jahrestag (1-366) eines Eingabedatums

FirstOfMonth

liefert das Datum des ersten Tages des Monats aus dem Eingabedatum

LastOfMonth

liefert das Datum des letzten Tages des Monats aus dem Eingabedatum

MMDDYYToDate

liefert ein Datum, basierend auf einem String im Format “MM/DD/YYYY”

Month

liefert die Monatszahl (1-12) eines Eingabedatums

MonthDay

liefert einen String im Format “MM/DD” basierend auf dem Eingabedatum

PopupDate

fragt den Benutzer nach einem Datum, unter Verwendung des Standard Dialogs

SubtractDays

subtrahiert die angegebene Anzahl Tage von dem Eingabedatum

SubtractMonths

subtrahiert die angegebene Anzahl Montage von dem Eingabedatum

SubtractYears

subtrahiert die angegebene Anzahl Jahre von dem Eingabedatum

ToDate

liefert ein Datum, basierend auf einem String im Format “YYYY/MM/DD”

Today

liefert das aktuelle Systemdatum

Year

liefert die Jahreszahl eines Eingabedatums

YearMonth

liefert einen String im Format “YY/MM” basierend auf dem Eingabedatum
