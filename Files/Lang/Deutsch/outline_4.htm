05.D Grafische Befehle

 

CreateWindow

erzeugt ein neues Fenster in dem Grafiken dargestellt werden können

DestroyWindow

entfernt ein vorher erzeugtes Fenster zum Anzeigen von Grafiken

DrawBitmap

zeichnet eine Bitmap Grafik auf die grafische Anzeige

DrawChars

zeichnet einen Textstring auf die grafische Anzeige

DrawLine

zeichnet eine Linie auf die grafische Anzeige

DrawRectangle

zeichnet ein Rechteck auf die grafische Anzeige

EraseWindow

löscht den Inhalt der grafische Anzeige

FillRectangle

füllt ein Rechteck mit einer benutzerdefinierten Farbe

SetCurrentWindow

setzt das aktuelle Grafikfenster, auf welches sich alle weiteren grafischen Befehle beziehen sollen
