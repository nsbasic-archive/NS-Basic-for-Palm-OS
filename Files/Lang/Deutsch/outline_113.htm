Left                                                                                                            Funktion

 

LEFT(theString as String, num as Integer)

 

Beschreibung

Liefert die in num angegebene Anzahl Zeichen des Strings theString von links aus gesehen. Wenn theString weniger als die verlangten Zeichen beinhaltet, so wird das Ergebnis der gesamten Stringlänge entsprechen.

 

Beispiel

Dim result as String

Result=Left(“John Doe”, 6) ‘liefert result=“John D”
