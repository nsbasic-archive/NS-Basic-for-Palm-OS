RTrim                                                                                                        Funktion

 

RTRIM(theString as String) As String

 

Beschreibung

Liefert einen String, in dem alle Leerzeichen am Ende entfernt wurden.

 

Beispiel

Dim result as String

Result=RTrim(CityName)

 

Scrollbar Objekt

Eine SCROLLBAR ist eine Leiste auf der der Anwender eine Position irgendwo zwischen dem Beginn und dem Ende der Leiste anwählen kann. Wird normalerweise genutzt um die Position innerhalb einer variablen Anzahl von Elementen zu finden. Wenn width größer ist als height, wird die Scrollbar horizontal dargestellt.

 

unterstützte Eigenschaften (während der Entwicklungsphase setzten)

Min Value, Max Value, PageSize, Value, Visible

 

Unterstützte Methoden (siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Min, Max, PageSize, Current, ID, Index, Type

 

Beispiel

Msgbox "Aktuelle Scrollbar Position ist : " + myScrollbar.Value
