Repeater                                                                                                       Objekt

 

Ein REPEATER Objekt einer Schaltfläche (Button Objekt) sehr ähnlich, außer dass der REPEATER markiert bleibt, solange der Anwender den Stift darauf gedrückt hält. Solange wird auch der mit dem Repeater verbundene Code immer wieder ausgeführt. Normalerweise wird dies verwendet, um, mittels des Symbolzeichensatzes, Pfeile nach links oder rechts als Text darzustellen. So kann, je nachdem auf welchen Button geklickt wird, ein Wert konstant erhöht, bzw. erniedrigt werden, und in einem separaten Textfeld dargestellt werden. Wird von eingebauten Anwendungen oft verwendet to kontinuierlich die Werte eines Datums (Jahr, Monat und Tag) zu erhöhen.

 

unterstützte Eigenschaften (während der Entwicklungsphase setzten)

Label, Font ID, Text, Anchor Left, Frame, Non-bold Frame, Visible

 

Unterstützte Methoden (siehe "Methoden")

Left, Top, Width, Height, Hide, Show, Redraw, Text, ID, Index, Type
