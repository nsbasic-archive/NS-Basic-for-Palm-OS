Round                                                                                                      Funktion

 

ROUND(theNumber as Float , pos as Integer) As Float

 

Beschreibung

Liefert eine Fließkommazahl, bei der theNumber auf die angegebene Nachkommastelle gerundet wird. Ist die nachfolgende Ziffer größer gleich 5, so wird aufgerundet, ansonsten wird der Rest abgeschnitten.

 

Beispiel

Dim result as Float

Result=Round(22.1256, 2) ‘liefert result=22.13

Result=Round(7.12312, 3) ‘liefert result=7.123
