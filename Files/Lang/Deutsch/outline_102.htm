Grid                                                                                                                Objekt

 

Mit dem Grid (Raster) Objekt können Sie Daten, die per Programm oder durch automatisches Laden aus einer Datenbank erzeugt wurden, in Tabellenform anzeigen. Der grundsätzliche Aufbau des Rasters wird während der Programmierung festgelegt und zur Laufzeit mit Daten gefüllt.

 

Wenn eine Zelle des Rasters angeklickt wird, werden die Zeilen- und Spaltenangaben bereitgestellt und können von Ihrem Programm ausgewertet werden. Zwar können die Zellen eines Grid nicht direkt vom Anwender bearbeitet werden, aber Sie können deren Inhalt mittels der Parameter TextMatrix und ValuesMatrix für Strings bzw. numerische Werte ändern. Text und Value können auf die gerade ausgewählte Zelle des Rasters angewendet werden.

 

Lesen Sie auch Abschnitt 9 dieses Handbuches, der eine detaillierte Beschreibung dieses Objektes enthält.

 

Spezifische Eigenschaften und Methoden dieses Objektes

 

.BINDTODATABASE dbName, dbFieldNameList [Where condition]

Füllt ein Grid automatisch mit Daten aus einer Datenbank. dbName ist der Variablenname einer Datenbank, der zuvor mittels DIM WITH RECORD Anweisung festgelegt wurde. dbFieldNameList enthält eine Liste von Feldnamen oder den Namen einer TYPE-Struktur der Datenbank. Condition ist ein optionales Argument, das festlegt, welche Datensätze angezeigt werden. Das Format entspricht dem einer IF Anweisung

.COL

Lese oder setze die aktuelle Spalte. Liegt zwischen 1 und COLS.

.COLS

Lese die Anzahl der Spalte.

.COLTYPE colNo, type, formatString

Setzt den Typ und das Format einer Spalte. Der Vorgabewert für Spalten ist Text ohne Inhalt. Wenn Sie den Datentyp einer Spalte ändern wollen, müssen Sie dazu zur Laufzeit diese Methode verwenden. ColNo ist eine existierende Spaltennummer im Raster. Type ist entweder "text","numeric" oder "checkbox". FormatString hängt von type ab: für "text", stellt es den Vorgabewert der Zelle dar, für "numeric", ist es ein Formatstring, wie er beim FORMAT Befehl verwendet wird. Bei "checkbox" wird der formatString rechts neben der Checkbox angezeigt.

.COLWIDTH(colNo)

Lese oder setze die Breite der Spalte columnNo. Zum Ausblenden einer Spalte auf -1 setzen.

.FONTMATRIX(rowNo, ColNo)

Lese oder setze die Font-Nummer (0-7) für eine Zelle.

.HIDEGRIDLINES

Gitterlinien ausblenden.

.ROW

Lese oder setze die aktuelle Zeilenummer. Liegt zwischen 1 und ROWS. Auf 0 setzen, um die Gitterauswahl zurückzusetzen.

.ROWDATA

Lese oder setze den Wert rowdata der Zeile. (Benutzerdefinierter Wert)

.ROWS

Lese oder setze die Anzahl der Zeilen.

.SHOWGRIDLINES

Gitterlinien anzeigen

.TEXT

Lese oder setze den Text der aktuellen Zelle.

.TEXTMATRIX(rowNo, ColNo)

Lese oder setze den Text der Zelle an Position rowNo, ColNo.

.TOPROW

Lese oder setze die erste anzuzeigende Zeile

.VALUE

Lese oder setze den numerischen Wert der aktuellen Zelle.

.VALUEMATRIX(rowNo, ColNo)

Lese oder setze den numerischen Wert der Zelle an Position rowNo, ColNo.

 

Unterstütze Eigenschaften (beim Programmieren festlegen)

Cols, Has Scrollbar, Height, Left, Top, Visible Rows, Width

 

Andere unterstütze Methoden (Siehe auch "Methoden")

Add, Clear, Hide, Redraw, Remove, Show
