Global                                                                                                            Befehl

 

Global varname as type

Global varArray(nnn) as type

Global varname as type* length[,decimalPlaces

Global varArray(nnn) as type*length[,decimalPlaces]

 

Beschreibung

Definiert eine Variable, die global gültig ist. Dadurch kann diese innerhalb aller Code Module in der gesamten Anwendung verwendet werden. Dies ist anders als die Verwendung eines DIM Ausdruckes, da solche Variabeln nur in jenen Funktionen oder Subroutinen verwendet werden können, wo Sie mittels DIM definiert wurden. Für weitere Informationen bezüglich des DIM Ausdruckes, sehen Sie bitte bei dem DIM Ausdruck nach. Das GLOBAL Statement wird normalerweise in Ihrem Startup Code verwendet.

 

Beispiel

Global amount as Float

Global warehouseNames(25) as String
