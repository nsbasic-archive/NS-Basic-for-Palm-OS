06.H Serial I/O Built-In Functions

 

SerialOpen

Opens the serial port and prepares for input/output

SerialClose

Closes the serial port to discontinue its use

SerialReceive

Accepts input from the serial port

SerialReceiveWithEvent

Sets serial communications to wait for input from serial port.

SerialSend

Transmits data out through the serial port

SerialSet

Sets the value of several serial port parameters to control transmission options

SerialDial

Dials a phone number through the modem

SerialHangup

Disconnects a phone line connection
