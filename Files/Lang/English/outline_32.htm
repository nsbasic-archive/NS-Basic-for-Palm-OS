Button                                                                                                           Object

A BUTTON is a rectangular box with rounded corners that contain text inside the box. Buttons are normally used to indicate a possible action to be taken. When the user taps on the button, the program code associated with that button is executed.

 

Properties Supported (Set at design time)

Left, Top, Width, Height, Label, Font ID, Anchor Left, Frame, Non-bold Frame, Visible

 

Methods Supported (See "Methods")

Hide, Show, Redraw, Label, ID, Index, Type
