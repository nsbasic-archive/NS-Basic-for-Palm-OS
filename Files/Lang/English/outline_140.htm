PopupTime                                                                                             Function

 

POPUPTIME(startTime as Time, endTime as Time, title as string)

 

Description

Shows the user starting and ending times and let the user change these times using the Palm OS device's standard popup for time inputs. If the user changes either time, the new times will be stored in startTime and endTime and the function will return a value of 1. If the user does not modify the times, then the values in startTime and endTime will not change and the function will return a value of 0. The title can be used to prompt the user for the type of times being requested. This function requires Palm OS 3.3 or later. On some devices, calling this function will cause the After Script to be run on return.

 

 

Example

Dim apptstart as time

Dim apptend as time

Dim res as integer

Res=PopupTime(apptStart, apptEnd , "Enter the times of your appointment")
