DbCreate                                                                                                 Function

 

DBCREATE(dbName as Database, fileName as String, cardNo as Integer, creator as String)

 

Description

Creates a new database on a specified memory card of the Palm OS device, and returns an integer result. See DBOPEN for a table of result codes.

 

DbName references the database inside the program. All actions on the database will identify it by this name. FileName is the name of the database as recorded on the Palm OS device Memory system. CardNo is 0 on all modern devices. Creator is a 4-character Creator ID used to identify the owner or creator of the database Normally, you will use the same Creator ID as you used for your project.. To avoid duplications, Palm Computing allows you to register your Creator ID on their website to insure it is unique and not duplicated by someone else. The file is created with the backup attribute set, which means that Hotsync will automatically back it up to your desktop. The Palm naming convention is to follow your database name with a hyphen and your Creator ID.

 

Example

Dim result as integer

Dim CustomerDb as database

Result=DbCreate(CustomerDb, "Company-Test", 0, "JimB")
