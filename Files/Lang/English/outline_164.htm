SerialReceiveWithEvent                                                                      Function

 

SERIALRECEIVEWITHEVENT()

 

Description

This function sets up the device to wait for data to come in, When characters arrive, an NsbSerialEvent is generated that you can pick up in your Event code. This function has no parameters. See SerialOpen for a list of result codes. Requires Palm OS 3.5 or later; not supported in all versions of Palm OS 5. A workaround for devices which do not have this feature is to use a timer to check the serial port regularly.

 

Example

'anywhere in your program after serial port is opened

Global err as integer

err=SerialReceiveWithEvent()'in your form's Event Script

   Dim received as String

    If getEventType()=nsbSerialIn Then

        MsgBox "Serial In Event"

        err=serialReceive(received, 10, 0)

      If Err > 0 Then

        Beep

        error.text=str(err)

        data.text=data.text+"."+Received

      Else

        error.text=""

        data.text=data.text+received

      End If

    End If
