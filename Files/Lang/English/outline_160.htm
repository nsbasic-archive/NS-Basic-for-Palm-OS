SerialDial                                                                                                 Function

 

SERIALDIAL(phoneNo as string, modemSetup as string)

 

Description

Dials the specified phone number after setting the modem using the

modem setup commands.  PhoneNo is the number to be dialed. ModemSetup is the command string to send to the modem to initialize it. Omit the leading 'AT' letters as these are automatically supplied. Returns an integer error code. See SerialOpen for a list of result codes.

 

Example

Err=SerialDial("13136213555","")
