DbReset                                                                                                   Function

 

DBRESET(dbName as Database)

 

Description

Resets the database so DBREADNEXT will read the first record in the database. The database must have been previously opened with DBOPEN. Returns an integer result. See DBOPEN for a table of result codes.

 

 

Example

Dim res as integer

Res=DbReset(CustomerDb)
